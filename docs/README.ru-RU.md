# Accessible OBS Studio 1.0

[![Логотип Tiflo.Info: голубые волны над названием Tiflo.info и слоганом «Закрой глаза и смотри»](../assets/tiflo-info-logo.jpg)](https://tiflo.info)

Accessible OBS Studio — плагин специальных возможностей для OBS Studio под Windows. Он предназначен для незрячих пользователей, работающих с клавиатурой и программой экранного доступа, в частности JAWS или NVDA.

## Требования и установка

Требуются 64-разрядная Windows 10 или 11 и поддерживаемая 64-разрядная версия OBS Studio 32. Собственный ключ API и доступ к Интернету нужны только для функций OpenAI.

Закройте OBS и запустите `AccessibleOBSStudio-1.0-Setup.exe`. Установщик разместит плагин в `C:\ProgramData\obs-studio\plugins\accessible-obs-studio` и проверит наличие Microsoft WebView2 и Visual C++. Недостающие компоненты загружаются только при необходимости, непосредственно с серверов Microsoft; для этого нужен доступ в Интернет. Файлы OBS и Qt не заменяются. Для удаления воспользуйтесь разделом Windows «Установленные приложения». Настройки по умолчанию сохраняются.

## Клавиши по умолчанию

- F3 — захватить холст и отправить его в OpenAI для анализа.
- F4 — перейти к отображаемой штатной панели управления мультимедиа.
- F5 — начать или остановить трансляцию.
- F6 / Shift+F6 — следующая / предыдущая основная область интерфейса.
- F7 — начать или остановить запись.
- F8 — включить или выключить виртуальную камеру.

Команда **Accessible OBS Studio: Open Accessible OBS Studio** открывает редактор сочетаний клавиш. Ей можно назначить сочетание, но по умолчанию оно не задано.
- Ctrl+0…Ctrl+5 — перейти к видеопросмотру, сценам, источникам, аудиомикшеру, переходам или элементам управления.

При первом запуске плагин отключает горячие клавиши OBS, когда главное окно OBS не в фокусе, но лишь в том случае, если пользователь ещё ни разу не задавал этот параметр самостоятельно. Все последующие изменения сохраняются.

## Редактор сочетаний клавиш

Откройте **Сервис > Accessible OBS Studio**. Найдите команду и нажмите Enter либо кнопку **Добавить или изменить**. Настройки OpenAI открываются кнопкой **API Settings** в этом окне. В диалоге «Hot Key» можно назначить команде несколько сочетаний. Кнопка **Добавить ещё одно сочетание** создаёт дополнительное назначение. Enter подтверждает изменения, Escape отменяет их. Tab, Shift+Tab, Enter, Escape, Alt+F4, сочетания с клавишей Windows и зарезервированные системные команды не перехватываются.

Изменения записываются в OBS только после нажатия OK в главном окне редактора. Delete или **Удалить** очищает все назначения выбранной команды. При закрытии окна с несохранёнными изменениями предлагаются варианты «Сохранить», «Не сохранять» и «Отмена».

## Аудиомикшер и мультимедиа

Ctrl+3 переводит фокус в аудиомикшер. Видимые регуляторы громкости нумеруются в порядке отображения. Клавиши 1–9 выбирают первые девять источников, а 0 — десятый. Стрелки и остальные команды работают штатным образом в OBS. F4 фокусирует панель управления мультимедиа, если OBS её отображает; плагин не подменяет встроенные мультимедийные клавиши OBS.

## Анализ холста и действия

Ключ API необязателен. Сохраните его только при необходимости кнопкой **API Settings** в редакторе сочетаний клавиш. В диалоге работают Tab и Shift+Tab; нажатие Enter в поле ключа активирует кнопку **Сохранить ключ**. Перед шифрованным сохранением в Диспетчере учётных данных Windows проверяются формат ключа и авторизация OpenAI. Без ключа блокируются только функции OpenAI; остальные возможности плагина продолжают работать.

Для F3 не требуется предварительно фокусировать окно предпросмотра. Короткий щелчок подтверждает, что запрос начат. Во время захвата фокус остаётся на текущем элементе. Затем фокус получает окно результата; после его закрытия плагин возвращается к прежнему элементу OBS, если тот ещё существует. Если пользователь перешёл в другую программу, плагин не забирает у неё фокус.

Уточняющие вопросы должны непосредственно относиться к захваченному изображению. Текст на изображении и сообщения пользователя считаются недоверенными данными. Посторонние вопросы отклоняются. Контекст удаляется при закрытии окна или новом захвате.

Кнопка **Предлагаемые действия** открывает ограниченный список обратимых преобразований OBS. Если источник не выбран, появляется доступный список источников. Каждое действие имеет отдельный флажок и уровень риска. До нажатия **Применить выбранное** ничего не выполняется. Используются штатные действия OBS и его журнал отмены. Управление трансляцией, записью, звуком, учётными данными, произвольными командами и параметрами вывода запрещено.

## Анализ совместимости

При запуске в непроверенной основной версии OBS предупреждение предлагает **Отмена**, **Всё равно запустить** и **Анализировать совместимость**. Анализ объединяет локальные проверки только для чтения с официальными источниками OBS и оценивает риск как низкий, средний, высокий или неизвестный. Это не гарантия совместимости.

Успешный отчёт сохраняется для точной версии OBS, версии плагина и архитектуры. В дальнейшем он открывается без нового обращения к OpenAI. Отчёт показывается в доступном окне WebView2 и автоматически копируется в буфер обмена.

## Конфиденциальность, стоимость и лицензия

Для анализа холста в OpenAI отправляются холст, язык OBS, неизменяемая защитная инструкция и допустимые уточняющие вопросы. Для проверки совместимости отправляются номера версий, сведения о зависимостях и фиксированный перечень возможностей, но не сцены, медиафайлы и не учётные данные. Расходы API оплачивает владелец ключа. Телеметрии и рекламы нет.
