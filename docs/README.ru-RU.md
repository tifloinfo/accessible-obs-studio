# Accessible OBS Studio 1.0.1

Accessible OBS Studio — плагин специальных возможностей для 64-разрядной OBS Studio 32 под Windows 10 и 11. Он предназначен для незрячих пользователей клавиатуры и экранного диктора и проверен с JAWS и NVDA. Ключ OpenAI API и Интернет нужны только для функций OpenAI.

## Установка

Установите 64-разрядную OBS Studio 32.0 или новее, затем запустите `AccessibleOBSStudio-1.0.1-Setup.exe`. Если OBS Studio не установлена, повреждена или имеет версию ниже 32.0, установщик предложит открыть [официальную страницу загрузки OBS](https://obsproject.com/download) и завершит работу без изменений. Старую версию также можно обновить командой «Справка > Проверить обновления» в OBS Studio. OBS Studio 32.x поддерживается. При наличии OBS Studio 33 или новее установщик предупредит о возможной несовместимости и предложит [страницу последней версии плагина](https://tiflo.info/aobs), прежде чем разрешить явно выбранную установку несмотря на предупреждение. Если OBS Studio запущена, установщик попросит полностью закрыть ее и нажать «Повторить»; автоматически OBS не завершается. Плагин устанавливается в `C:\ProgramData\obs-studio\plugins\accessible-obs-studio`. Недостающие компоненты Microsoft WebView2 и Visual C++ добавляются только после этих проверок; файлы OBS и Qt не заменяются. На заключительной странице флажок **Открыть файл ReadMe в веб-браузере** открывает русскую HTML-документацию.

## Сочетания клавиш по умолчанию

- F3: краткое описание холста, не более 80 символов.
- Shift+F3: подробное описание.
- Alt+F3: чтение видимого текста без перевода и комментариев.
- Ctrl+F3: описание людей и фона.
- F4: Визуальная проверка трансляции или записи на проблемы компоновки, камеры, освещения, кадрирования, резкости, зернистости, внешнего вида, одежды, фона и посторонних предметов.
- Ctrl+M: фокус на видимых элементах управления медиапроигрывателем.
- F5, F7, F8: начать или остановить трансляцию, запись или виртуальную камеру.
- F6 / Shift+F6: следующая / предыдущая основная область.
- Ctrl+0–Ctrl+5: холст, сцены, источники, аудиомикшер, переходы или управление.
- Ctrl+` (клавиша под Escape): открыть доступную консоль громкости.

Команда **Accessible OBS Studio: Открыть редактор сочетаний клавиш** напрямую открывает тот же редактор и не имеет стандартного назначения. Её внутренний идентификатор не изменился, поэтому существующее назначение сохраняется.

По умолчанию Accessible OBS Studio принудительно разрешает все сочетания клавиш OBS только тогда, когда OBS является активным приложением. Параметр **Настройки > Расширенные > Поведение сочетаний клавиш при смене фокуса** поддерживается в состоянии **Отключать сочетания клавиш, когда главное окно не в фокусе** и восстанавливается после изменений. Чтобы вернуть управление OBS, отметьте и сохраните в редакторе параметр **Разрешить OBS Studio управлять работой сочетаний клавиш вне OBS**. После этого плагин перестанет вмешиваться.

При первом запуске и после смены профиля плагин сравнивает свои запланированные назначения с существующими. Диалог появляется только при реальном конфликте. Можно удалить только конфликтующие назначения и применить стандартные сочетания Accessibility либо сохранить существующие назначения; в этом случае конфликтующие стандартные сочетания останутся неназначенными. Параметр **Больше не спрашивать для этой версии** действует во всех профилях, но сбрасывается новой версией или сборкой.

## Редактор сочетаний клавиш

Откройте **Сервис > Accessible OBS Studio**, чтобы напрямую открыть редактор сочетаний клавиш. Стрелки перемещают по списку команд, Tab — между выбранной командой, по умолчанию снятым флажком управления сочетаниями OBS и кнопками. Enter или **Добавить или изменить** открывает диалог назначения. Enter и OK немедленно проверяют дубликаты. При конфликте называется другая команда: Нет возвращает к вводу, Да переназначает сочетание. Для настройки OpenAI нажмите в редакторе кнопку **Настройки API OpenAI**.

## Микшер и медиапроигрыватель

Ctrl+3 фокусирует обычный микшер OBS. Плагин больше не нумерует его регуляторы и не устанавливает на них глобальный фильтр событий. Ctrl+` открывает модальную доступную консоль: стрелки влево и вправо выбирают источник, вверх и вниз изменяют громкость на 1 дБ, Home устанавливает 0 дБ, пробел включает или выключает звук, цифры 1–0 выбирают первые десять источников.

При открытии консоль получает исключительное управление доступными в этот момент источниками, уровнями громкости и состояниями выключения звука. Изменения немедленно применяются в OBS, но во время сеанса консоль не отслеживает изменения через обычный микшер, внешние контроллеры, переключение сцен или другие способы управления. Не изменяйте микшер другим способом, пока консоль открыта; закройте и снова откройте её, чтобы загрузить внешние изменения.

## Описание холста

Все пять режимов захватывают отрисованный холст OBS. Только самый новый ответ имеет агрессивную роль ARIA alert; вопрос пользователя никогда не повторяется. Во всех режимах доступны уточняющие вопросы.

В окне краткого описания кнопка **Подробное описание** доступна всегда, **Прочитать текст** — только при найденном тексте, **Люди и фон** — только при найденных людях, а **Предлагаемые исправления** — только при наличии проблемы, которую действительно можно исправить автоматически. Эти команды используют уже отправленное изображение.

Режим **Люди и фон** в первую очередь описывает видимых людей, а затем их непосредственный фон. Несвязанные элементы интерфейса, текст и детали сцены опускаются, если они напрямую не влияют на представление человека.

**Визуальная проверка** оценивает только внешний вид трансляции или записи. Сохраняются все визуальные проверки компоновки OBS, пустого захвата, камеры, освещения, полноэкранного режима Zoom, кадрирования, размытия, возможного загрязнения объектива, зернистости, внешнего вида, одежды, фона и посторонних предметов. Словесное содержание игнорируется: язык, орфография, грамматика, перевод, формулировки, факты, числа, тема, тон, уместность, субтитры и подписи не считаются проблемами. Текст может быть отмечен только как визуальный объект, если он слишком мелкий, обрезан, размыт, малоконтрастен, заслонён или сам закрывает важные элементы изображения. Диалоговое окно или сообщение об ошибке отмечается только тогда, когда оно закрывает содержимое или показывает видимую проблему захвата либо компоновки, но никогда из-за смысла сообщения. Автоматическое исправление предлагается только для фиксированного списка обратимых преобразований OBS. **Проверить снова** делает новый снимок и сообщает об улучшениях, ухудшениях, изменениях и оставшихся визуальных проблемах.

Ключ API проверяется перед сохранением, хранится в Диспетчере учётных данных Windows и никогда не отображается. Его удаление требует подтверждения и сопровождается сообщением об успехе.
